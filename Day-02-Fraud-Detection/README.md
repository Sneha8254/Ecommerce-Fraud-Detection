# E-Commerce Fraud Simulation & Detection 🚀

An end-to-end self-initiated project focused on synthesizing and analyzing financial transactions to detect fraudulent patterns. 

## 📌 Project Overview
- **Day 1:** Built a data simulation factory using Python, Faker, and Pandas to generate 10,000 realistic e-commerce transactions.
- **Anomalies Injected:** Implemented localized fraud logic to simulate identity theft, high-value transaction spikes, and geographical shipping/billing mismatches.
- **Current Dataset:** Successfully generated `raw_transactions.csv` containing 166 flagged fraudulent transactions (1.66% anomaly rate).

## 🛠️ Tech Stack (Day 1)
- **Language:** Python 3.13
- **Libraries:** Pandas, Faker, Random, CSV
- **Environment:** Mac Terminal / Zsh / Nano
