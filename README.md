# 🛡️ E-Commerce Fraud Detection Pipeline

A self-initiated 7-day project building an end-to-end e-commerce fraud simulation and data detection pipeline using Python and Pandas.

## 📊 Project Progress

### 🔹 Day 1 & Day 2: Data Generation & Initial Flagging
* Built a simulation engine (`generate_data.py`) to mimic e-commerce transaction logs.
* Developed a rule-based detection script (`detect_fraud.py`) to flag high-risk transactions based on order thresholds and risk factors, exporting the results to a structured dataset.

### 🔹 Day 3: Data Analytics & Financial Intelligence (Current Milestone)
* Created an analytical reporting engine (`analyze_fraud.py`) utilizing advanced Pandas aggregations.
* Successfully isolated and calculated key financial risk metrics from the flagged transaction subset.

#### 📈 Day 3 Analytical Report Output:
* **Total Financial Amount at Risk:** \$149,014.27
* **Average Fraudulent Order Value:** \$892.30
* **Top Shipping Hotspots by Volume:** Mali (ML), Iraq (IQ), Austria (AT), Palau (PW), and Dominican Republic (DO).
